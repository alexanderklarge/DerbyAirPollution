Extra information about the data

The DEFRA Interactive monitoring networks map is pictured below. There is a sole sensor in Derby, located as the St Alkmund's bridge, as shown in the second screenshot.

The data from each sensor can be downloaded as raw CSV files, as shown below. This allows analysis in Microsoft Excel, but daily re-downloading of the files would be needed to get up-to-date data. Python, a programming language, can be used to automatically pull the data from the csv files, clean the data (i.e. remove unwanted columns) and create graphs which update each week.

The data is hourly. Below is the plot of a single day. You can clearly see at peak around the time most people drive to work and later when they return. You can "resample" the data to be daily, weekly etc.
